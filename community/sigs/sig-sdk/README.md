# SIG SDK

The SDK Special Interest Group owns the language-specific SDKs and developer experience for AI-SDLC implementors.

## Scope

- TypeScript SDK (`sdk-typescript/`)
- Python SDK (`sdk-python/`)
- Go SDK (`sdk-go/`)
- Reference implementation (`reference/`)
- Developer experience, documentation, and examples (`docs/`)

## Meetings

- **Cadence**: Bi-weekly
- **Notes**: Published in [`community/meetings/`](../../meetings/)
- **Agenda**: Posted 24 hours in advance

## Responsibilities

- Maintain language-specific SDK packages
- Keep SDKs in sync with specification changes
- Ensure consistent developer experience across languages
- Maintain the reference implementation
- Guide type generation from JSON schemas
