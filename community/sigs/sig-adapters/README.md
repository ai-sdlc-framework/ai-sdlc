# SIG Adapters

The Adapters Special Interest Group owns the adapter layer specification, interface contracts, adapter registry, and distribution builder.

## Scope

- Adapter layer specification (`spec/adapters.md`)
- Interface contract definitions (IssueTracker, SourceControl, CIPipeline, CodeAnalysis, Messenger, DeploymentTarget)
- Adapter registration and discovery process
- Distribution builder specification
- Community adapter reviews (`contrib/adapters/`)

## Meetings

- **Cadence**: Bi-weekly
- **Notes**: Published in [`community/meetings/`](../../meetings/)
- **Agenda**: Posted 24 hours in advance

## Responsibilities

- Define and evolve adapter interface contracts
- Review adapter implementations for conformance
- Maintain the adapter registration process and metadata format
- Guide community adapter contributions
