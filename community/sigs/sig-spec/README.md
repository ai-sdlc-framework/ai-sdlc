# SIG Spec

The Specification Special Interest Group owns the core specification documents, resource model, JSON schemas, and versioning strategy.

## Scope

- Core specification (`spec/spec.md`)
- Resource model and all core resource type definitions
- JSON Schema definitions (`spec/schemas/`)
- Versioning strategy and API maturity progression
- Conformance level definitions

## Meetings

- **Cadence**: Bi-weekly
- **Notes**: Published in [`community/meetings/`](../../meetings/)
- **Agenda**: Posted 24 hours in advance

## Responsibilities

- Review RFCs affecting core resource types
- Maintain JSON schemas and ensure consistency with normative text
- Define and evolve conformance levels
- Coordinate with other SIGs on cross-cutting concerns
