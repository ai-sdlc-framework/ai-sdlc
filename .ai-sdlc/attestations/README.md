# `.ai-sdlc/attestations/`

Per-commit DSSE envelopes generated by `/ai-sdlc execute` Step 10 (AISDLC-74).
Each file is named `<head-sha>.dsse.json` and committed alongside the work in
the chore commit so CI's `verify-attestation.yml` workflow can read it on
`pull_request`.

Old attestations stay (audit trail, ~1-2KB each). They are NOT secrets — the
private signing key never leaves contributor machines (`~/.ai-sdlc/signing-key.pem`).

Schema: `.ai-sdlc/schemas/attestation.v1.schema.json`. Trusted public keys:
`.ai-sdlc/trusted-reviewers.yaml`.
